# CascadeRule-LLM: Experiment Code & Dataset

This repository contains the code and data to reproduce all experiments in:

> **CascadeRule-LLM: Hierarchical Adaptive Weight Learning for Entity Resolution in Knowledge Graphs**

## Structure

```
├── code/                           # Experiment scripts
│   ├── l1_scorer.py                # L1 bidirectional rule scorer (20 signals, λ=1.5)
│   ├── config.py                   # API endpoints, model names
│   ├── l2_multimodel_experiment.py # L2 multi-model comparison (GLM-5/4.5/4.5-Air, Qwen3.6)
│   ├── feedback_loop_experiment.py # 5-round feedback convergence
│   ├── classifier_ablation_lr_rf.py# LR vs RF 5-fold CV ablation
│   ├── per_type_accuracy.py        # Per-type accuracy (5 types × 4 models)
│   ├── per_type_threshold_analysis.py
│   ├── per_type_lr_cv.py           # Per-type LR with strict CV
│   ├── hybrid_strategy.py          # Hybrid per-type with fallback (min_samples=50)
│   ├── per_type_feature_importance.py
│   └── cross_model_validation.py   # GPT-5.4 annotation (Cohen's κ)
├── data/
│   ├── dataset_v3_cleaned.json     # 2,639 candidate pairs with signals + labels
│   └── entity_fragments.json       # Source text fragments for L2 grounding
└── results/
    ├── l1_results_all.json         # L1 scores for all pairs
    ├── l2_v3_comparison.json       # L2 model comparison summary
    ├── feedback_loop_experiment.json
    ├── gpt54_annotation_300.json   # GPT-5.4 cross-model annotation
    ├── compiled_results.json
    └── checkpoints_v3/             # Per-model L2 outputs (5 models)
```

## Dataset

**`dataset_v3_cleaned.json`** — 2,639 candidate entity pairs from USN Monitor:

- 5 entity types: Equipment, Event, Location, Person, Organization
- Each pair has: entity names, VIDs, entity type, 20-dim signal features, expert label (same/different)
- Labels generated by GLM-5.2 (99.8% validity rate)
- Cross-validated: GPT-5.4 κ=0.788, human 3-way accuracy 88.2%

## Reproduction

```bash
# 1. L1 scoring
python code/l1_scorer.py

# 2. L2 multi-model experiment (requires API access)
python code/l2_multimodel_experiment.py

# 3. Feedback loop convergence
python code/feedback_loop_experiment.py

# 4. Classifier ablation
python code/classifier_ablation_lr_rf.py

# 5. Per-type analysis & hybrid strategy
python code/per_type_accuracy.py
python code/hybrid_strategy.py

# 6. Cross-model validation
python code/cross_model_validation.py
```

## Requirements

- Python 3.11+
- scikit-learn, numpy, pandas
- GLM API access (zhipuai) or compatible OpenAI-format endpoint
- Optional: Qwen3.6 local vLLM server
